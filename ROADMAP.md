# Roadmap

## Phase 1: Offline Scanner Foundation

- Rust workspace and CLI.
- Deterministic prompt, jailbreak, secret, MCP, agent, and workflow analysis.
- Low-noise structured AI config parsing for Claude Code, Cursor, OpenAI Agents, LangGraph, CrewAI, and AutoGen.
- YAML rule engine.
- Terminal, JSON, SARIF, and HTML output.
- GitHub Action and CI.

## Phase 2: Assisted Analysis

- Optional LLM-assisted triage behind explicit opt-in.
- AI-generated remediation suggestions with source redaction.
- Attack simulation fixtures for prompt and tool abuse.

## Phase 3: Runtime Inspection

- Live MCP server inspection.
- Agent behavior auditing.
- Runtime tool invocation policy checks.

## Phase 4: Enterprise Operations

- Web dashboard.
- Team and policy management.
- Multi-repository scan aggregation.
- Historical trend reporting.
